# Fomofly 使用 SOP — 衛教講座自動剪輯

## 什麼是 Fomofly？
台灣 AI 影片剪輯工具，上傳長影片後自動識別精華片段，
一鍵剪成適合各平台的短影片（Reels/Shorts/TikTok）。

---

## 使用流程（衛教講座 → 短影片）

### Step 1：準備素材
- 錄好衛教講座（Zoom/實體，建議 15-30 分鐘）
- 確保音質清晰（用麥克風，不要用手機內建）
- 上傳到 Google Drive 備用

### Step 2：上傳到 Fomofly
1. 前往 [fomofly.tw](https://fomofly.tw)
2. 建立新專案，上傳影片
3. 選擇目標平台（IG Reels / TikTok / YouTube Shorts）
4. 等待 AI 分析（約 5-10 分鐘）

### Step 3：審核自動選出的片段
AI 會選出：
- 觀點強烈的句子（適合做開頭）
- 數字/案例片段（高留存率）
- 結尾行動號召

**重要：** 每個片段都要人工確認內容正確性！

### Step 4：搭配字幕和 B-Roll
- 開啟自動字幕功能
- 手動校正專業術語（血糖、胰島素等）
- 上傳食物照片作為 B-Roll

### Step 5：搭配本 Toolkit
```
先用 Fomofly 剪片 → 再用本系統 功能5（✂️ 內容改寫）
生成配合短影片的文字貼文
```
